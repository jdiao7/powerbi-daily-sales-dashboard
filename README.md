# Power BI Daily Sales Dashboard

This project analyzes daily sales data from a **Daily coffee chain location in the Philippines** using Power BI.  
The goal of the dashboard is to transform raw, operational sales data into clear, business-focused insights for store owners and managers.

---

## Project overview

The raw dataset consisted of daily transactional sales records with over **200 columns**, many of which were inconsistent, redundant, or not useful for decision-making.

This project focused on:
- Cleaning and restructuring messy real-world data
- Designing a data model suitable for analytics
- Building intuitive dashboards that highlight key performance indicators (KPIs)

---

## Data preparation (Power Query)

Key data preparation steps included:
- Removing unused and redundant columns
- Standardizing date and time fields
- Handling missing and inconsistent values
- Restructuring the dataset to support KPI-level analysis
- Creating a clean fact table optimized for reporting

Power Query was used extensively to ensure the final dataset was reliable and analysis-ready.

---

## Data modeling & measures (DAX)

- Built a dedicated **date table** to support time-based analysis
- Created DAX measures for:
  - Total sales
  - Daily and monthly trends
  - Performance comparisons over time
- Ensured relationships supported accurate filtering and aggregation

---

## Dashboard features

The dashboard provides:
- High-level sales KPIs for quick performance assessment
- Daily and monthly sales trends
- Visuals designed for non-technical business users
- A clean layout focused on clarity and usability

The dashboard was designed with real business stakeholders in mind, prioritizing simplicity and actionable insights.
